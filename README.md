Proyecto: Food Store
Sistema para la gestión de pedidos y productos desarrollado en Java.

Cómo ponerlo en marcha:

Base de Datos:

Abrí MySQL Workbench.

Importá el archivo sqlfood.sql usando la opción "Data Import/Restore".

Esto creará la base de datos food_store_db con todas las tablas necesarias.

Ejecución en NetBeans:

Abrí el proyecto en Apache NetBeans.

Hacé clic derecho en el proyecto y seleccioná Clean and Build.

Ejecutá el archivo Main.java para ver el funcionamiento del sistema.
